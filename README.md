# Ising-model-simulation
El modelo de Ising es un sistema de spins en una red donde cada elemento interactúa con sus vecinos, utilizado para estudiar fenómenos como el magnetismo y transiciones de fase.

Herramientas:

-Simulación mediante algoritmo de Monte Carlo (Metropolis)

-Python (NumPy, Matplotlib)


Resultados:

-Observación de transición de fase

-Cambio en magnetización al variar la temperatura

-Comportamiento colectivo emergente


Conclusión:

-El modelo reproduce comportamiento típico de sistemas físicos complejos y permite analizar cómo interacciones locales generan propiedades macroscópicas.


Data Perspective:

-Simulación de datasets complejos a partir de reglas locales

-Análisis de patrones emergentes

-Modelado computacional de sistemas

-Exploración de variables (temperatura vs magnetización)
